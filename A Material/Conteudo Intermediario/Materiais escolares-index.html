Lista de Materiais Escolares

Abaixo está uma lista de materiais escolares comuns, suas traduções e exemplos de uso.

Material (Inglês) Tradução Exemplo Ações
Pencil Lápis I need a pencil to write.
Notebook Caderno I use a notebook to take notes.
Eraser Apagador The eraser helps to fix mistakes.
Backpack Bolsa I carry my books in a backpack.
Ruler Regua The ruler is used to measure.
Calculator Calculadora I use a calculator for math.
Scissors Tesoura Scissors are used for cutting paper.
Glue Cola I need glue for my art project.
Marker Caneta Markers are great for coloring.
Folder Pasta I keep my papers in a folder.
O "pencil" é uma ferramenta de escrita que pode ser apagada, ideal para rascunhos.
Um "notebook" é usado para anotações e organização de matérias escolares.
Um "eraser" é usado para apagar marcas feitas por lápis.
Uma "backpack" é utilizada para carregar materiais escolares e pessoais.
Uma "ruler" é uma ferramenta de medição e também pode ser usada para desenhar linhas retas.
Uma "calculator" é usada para realizar operações matemáticas rapidamente.
As "scissors" são usadas para cortar papel e outros materiais.
A "glue" é uma substância adesiva que une materiais.
Os "markers" são utilizados para escrever ou colorir, disponíveis em várias cores.
Uma "folder" é usada para organizar e proteger papéis e documentos.

Dicas sobre Materiais Escolares

  • Organização: Mantenha seus materiais organizados para facilitar o acesso durante as aulas.
  • Manutenção: Cuide dos seus materiais para que durem mais tempo.
  • Listas de Materiais: Antes do início do ano letivo, verifique a lista de materiais que a escola fornece.
  • Customização: Personalize seu material para torná-lo único e divertido.
